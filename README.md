Project Title: Arzaq ERP โ Scrap Management System

Project Overview: A specialized Enterprise Resource Planning (ERP) solution developed to digitize and streamline operations for the scrap metal and recycling industry. This Single Page Application (SPA) automates the entire business workflow, from weighbridge operations to financial reporting, ensuring data accuracy and operational efficiency.

Key Achievements & Technical Highlights:

Automated Weighbridge Logic: Engineered a dynamic calculation system for gross/tare weights, deductions, and real-time pricing for both buying and selling transactions.

Financial Module: Implemented a full accounting cycle including treasury management, expense tracking, and real-time Income Statement (P&L) generation.

Document Generation: Integrated html2canvas and jsPDF to generate, preview, and print professional invoices and financial receipts dynamically.

Security & Access Control: Built a role-based authentication system (RBAC) with granular permissions and a comprehensive Audit Log to track deleted transactions and ensure accountability.

Tech Stack: Developed using React.js for state management, tailored with Tailwind CSS for a responsive Arabic (RTL) interface, and utilized LocalStorage for data persistence logic.
